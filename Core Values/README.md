# Core Values
The core values of the computer science professional are: honesty, compassion, responsibility, and excellence.

## Honesty
To be truthful, earnest, and always act with integrity.
[Read more](<Honesty.md>)


## Compassion
To act and speak in the best interests of others.
[Read more](<Compassion.md>)

## Responsibility
To be aligned in word and action with this code of ethics. To know and follow local, regional, and national regulations with respect to computer science.
[Read more](<Responsibility.md>)

## Excellence
To be an eternal learner, applying the highest level of appropriate rigour in all things.
